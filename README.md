## installation
```
npm i geeksay 
```

## usage 
##### module
```
$ npm i geeksay 
const geeksay = require('geeksay');
console.log( geeksay(`I love my home`) );  // I <3 my 127.0.0.1
```

##### command-line
```
$ npm i geeksay 
$ npm link
$ geeksay I love my home
```
